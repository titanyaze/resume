  About me  

Hi, my name is Abid Asab, currently a student studying computer science in university. Having a passion for computer science I have been working on my own personal projects, these going from the ranges of computer games made with the game engine Unity 3D with C# to creating web applications using Node.js. Having an interest in programming and games my journey begins in early ages from as old as 14 years old when I worked on my first project which was to make a paintball simulator within Unity 3D. In which during those early years I have learnt quite a lot of how games and overall how computer generally work altogether.
Acquiring other skills along the way related to computer science such as working with Photoshop and other image editors such as Affinity Photo and GIMP. On top of that working with 3D Toolkits such as Autodesk Maya and Blender in which I am to create my own 3D models which have been used in the games that I have created. This goes on further in which during this time I was learning game Logic with C# and design patterns that are used in games such as object pooling. Currently I am improving my web development skills and getting more familiar with making web applications as this is one the fields that I hold an interest in and will find useful later in my computer science career.
